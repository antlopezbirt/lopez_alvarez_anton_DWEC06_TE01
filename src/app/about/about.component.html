Acerca de esta web

El apagón general del pasado 28 de abril suscitó muchas especulaciones sobre nuestro sistema eléctrico.

Esta web pretende divulgar información sobre dicho sistema de manera sencilla para para que cualquier persona pueda estar medianamente informada sobre los datos producidos por el sistema de Red Eléctrica (Redeia).

Si quieres opinar (siempre con respeto) no dudes en dejar un comentario en cualquiera de las secciones, incluida esta.
